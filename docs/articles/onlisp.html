论 Lisp

阅读原文

《论 Lisp》是对 Lisp 高级编程技巧的全面研究，以自底向上编程作为贯穿全书的主题。它首次完整地介绍了宏（macros）及其各种应用。同时，本书还涵盖了与自底向上编程相关的若干重要主题，包括函数式编程、快速原型开发、交互式开发以及嵌入式语言。最后一章对面向对象编程进行了比以往 Lisp 书籍更深入的探讨，逐步构建了一个可运行的 Common Lisp Object System（CLOS）工作模型。

《论 Lisp》不仅是一本不可或缺的参考书，也是一座软件资源库。书中的示例构成了一个函数和宏的代码库，读者可以直接将其运用到自己的 Lisp 程序中。

Prentice Hall，1993 年，432 页，平装。ISBN 0130305529。
