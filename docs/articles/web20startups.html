为顶级“Web 2.0”初创公司投票

April 2006 · 阅读原文

2006 年 4 月

几天前，我向 Web 2.0 大会的某位组织者建议，今年应该免费邀请一些小型初创公司。具有讽刺意味的是，去年参会的初创公司很少，因为会议费用太高，它们根本负担不起。

他说他们也正考虑这么做，并问我认为应该邀请哪些公司。当我试图列出一份名单时，突然意识到有一个更符合“Web 2.0”精神的方式：让大家一起来投票。

于是我请 reddit 的几位朋友为此专门搭建了一个子版块：web2.reddit.com。请大家提交并为你最喜欢的初创公司投票。（你可以用每个名字旁边的箭头进行投票。投票需要注册，但注册只需设置一个用户名和密码，无需提供邮箱地址。）

请注意，这是一份初创公司名单，而不是具体应用产品。你应该提名 37signals，而不是具体提 Basecamp。

这份榜单与大会没有任何官方关联。它只是一次实验，目的是看看哪些 Web 2.0 初创公司最受推崇。
