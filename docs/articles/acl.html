ANSI Common Lisp

阅读原文

《ANSI Common Lisp》既是一本Lisp编程的入门指南，也是一份便捷且最新的ANSI Common Lisp参考手册。初学者会发现书中细致的讲解和有趣的示例让Lisp编程变得容易上手。专业程序员则会欣赏它全面而务实的风格。

Prentice Hall出版社，1995年，432页，平装本。ISBN 0133708756。

可在 SoftPro（我的朋友经营的书店，非常靠谱）或 Amazon 购买。

这是一本最新的ANSI Common Lisp参考手册。对面向对象编程进行了深入探讨，详细解释了Common Lisp Object System（CLOS），并展示了如何编写自己的面向对象语言。书中包含20多个大型示例程序，涵盖光线追踪、文本生成、模式匹配、逻辑推理、生成HTML、排序与搜索、文件I/O、压缩以及日期运算等内容。特别注重对关键概念的讲解，包括前缀语法、代码与数据的区别、递归、函数式编程、类型、隐式指针、动态分配、闭包、宏、类优先级，以及泛型函数与消息传递的对比。提供完整的程序优化指南。对宏的解释是所有入门书籍中最清晰、最透彻的。示例程序展示了Lisp的各种编程风格，包括快速原型开发、自底向上编程、面向对象编程以及嵌入式语言。附录部分专门讨论调试，并列举了常见的错误类型。

“这是一本直截了当、文笔流畅的ANSI Common Lisp初级与中级教程兼参考书。它远不止是一本入门读物——得益于其详尽的参考部分，对大多数读者来说，它可以作为Steele那本书的一个实用替代品。”

- Richard Fateman，加州大学伯克利分校
