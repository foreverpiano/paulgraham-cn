微软已死

April 2007 · 阅读原文

2007 年 4 月

几天前，我突然意识到微软已经死了。当时我正在和一位年轻的创业者聊天，聊 Google 和 Yahoo 有何不同。我说 Yahoo 从一开始就被对微软的恐惧所扭曲，这就是为什么他们把自己定位成一家“媒体公司”而不是技术公司。然后我看着他的脸，发现他并不理解。那感觉就像我告诉他 80 年代中期女孩们有多喜欢 Barry Manilow 一样。Barry 谁？

微软？他什么都没说，但我看得出他不太相信居然有人会害怕他们。

从 80 年代后期开始，微软在软件世界投下了近 20 年的阴影。我还记得在那之前是 IBM。我基本上无视这个阴影。我从不使用微软的软件，所以它只是间接影响到我——比如我收到的来自僵尸网络的垃圾邮件。正因为我没怎么注意，所以当阴影消失时我也没有察觉。

但现在它确实不在了。我能感觉到。甚至已经没有人再害怕微软了。他们仍然赚很多钱——顺便说，IBM 也一样。但他们已经不再危险。

微软是什么时候死的，又是因为什么死的呢？我知道他们直到 2001 年看起来还很危险，因为我当时写了一篇 另一条前路，说他们其实没看起来那么危险。我猜他们到 2005 年就已经死了。我知道我们创办 Y Combinator 时，从没把微软当作我们所资助的初创公司的竞争对手。事实上，我们甚至从未邀请他们参加我们为初创公司组织的投资人演示日。我们会邀请 Yahoo、Google 和其他一些互联网公司，但我们从未费心邀请微软。他们那边也从未有人给我们发过邮件。他们活在另一个世界里。

是什么杀了他们？我认为有四件事，都发生在 2000 年代中期，而且几乎是同时发生的。

最明显的是 Google。一个镇上只能有一个老大，而他们显然就是那个老大。Google 现在无疑是最危险的公司，无论从好的还是坏的意义上来说都是。微软最多只能在后面一瘸一拐地跟着。

Google 是什么时候取得领先的？人们可能会把时间推回到 2004 年 8 月他们 IPO 的时候，但那时他们还没有主导讨论。我认为他们是在 2005 年取得领先的。Gmail 是让他们跨越那道界限的关键因素之一。它表明 Google 能做的远不止搜索。

Gmail 还展示了如果你利用后来被称为“Ajax”的技术，基于 Web 的软件能做到什么程度。而这正是微软死亡的第二个原因：所有人都能看到桌面时代已经结束了。现在看来，应用程序将运行在 Web 上似乎已是必然——不仅仅是邮件，而是所有东西，一直到 Photoshop。即使是微软现在也意识到了这一点。

具有讽刺意味的是，微软无意中帮助创造了 Ajax。Ajax 中的 x 来自 XMLHttpRequest 对象，它允许浏览器在显示页面的同时在后台与服务器通信。（最初与服务器通信的唯一方式是请求一个新页面。）XMLHttpRequest 是微软在 90 年代后期创建的，因为他们需要它来做 Outlook。他们没有意识到的是，这个东西对很多其他人也会非常有用——事实上，对任何想让 Web 应用像桌面应用一样工作的人都有用。

Ajax 的另一个关键组成部分是 Javascript，这是在浏览器中运行的编程语言。微软看到了 Javascript 的危险，并试图尽可能长时间地让它保持残缺。[[1]] 但最终开源世界赢了，他们开发出了 Javascript 库，这些库像树木在带刺铁丝网上生长一样，绕过了 Explorer 的残缺。

微软死亡的第三个原因是宽带互联网。现在任何在意这件事的人都可以拥有快速的互联网连接。而管道越大，你对桌面的需求就越小。

最后一根钉子来自——谁都没想到——苹果。多亏了 OS X，苹果以一种在科技行业极其罕见的方式起死回生。[[2]] 他们的胜利是如此彻底，以至于我现在看到一台运行 Windows 的电脑都会感到惊讶。我们在 Y Combinator 资助的人几乎全都使用苹果笔记本。在创业学校听众席上也是如此。现在所有懂电脑的人都用 Mac 或 Linux。Windows 是给老奶奶用的，就像 90 年代的 Mac 一样。所以不仅桌面不再重要，而且关心电脑的人反正也不会再用微软的东西了。

当然，苹果在音乐领域也把微软逼得节节败退，电视和手机也即将跟上。

我很高兴微软死了。他们就像尼禄或康茂德——只有继承来的权力才能把人变成那种邪恶。因为记住，微软的垄断并不是从微软开始的。他们是从 IBM 那里接过来的。从大约 1950 年代中期到 2005 年左右，软件行业一直被一个垄断所笼罩。几乎是其整个存续期间。“Web 2.0”之所以带有如此强烈的兴奋感，部分原因就是人们（无论是否有意识）感觉到这个垄断时代可能终于要结束了。

当然，作为一个黑客，我忍不住会想怎么修复破损的东西。微软还有可能卷土重来吗？原则上，是有可能的。要想明白怎么做，可以设想两件事：(a) 微软现在手上有多少现金，以及 (b) 十年前 Larry 和 Sergey 挨个拜访所有搜索引擎公司，试图以一百万美元的价格出售 Google 的想法，却被所有人拒绝。

令人惊讶的事实是，以微软这样一家富有公司的标准来看，顶尖的黑客——危险的顶尖黑客——可以非常便宜地获得。他们已经没法 招聘已过时 聪明人了，但他们其实只需要多花一个数量级的钱就能买到想要的全部。所以如果他们想再次成为竞争者，可以这样做：

买下所有优秀的“Web 2.0”初创公司。他们可以用比收购 Facebook 少得多的钱把其中大部分都买下来。

把他们全都放在硅谷的一栋大楼里，用铅屏蔽包围起来，保护他们不与 Redmond 有任何接触。

我觉得这么说很安全，因为他们永远不会这么做。微软最大的弱点是他们仍然没有意识到自己有多糟糕。他们仍然认为自己可以在内部写软件。也许按照桌面世界的标准，他们确实可以。但那个世界几年前就已经结束了。

我已经知道这篇文章会引发什么反应。一半读者会说微软仍然是一家利润极其丰厚的公司，我应该更谨慎，不要基于我们这个封闭的“Web 2.0”小圈子里少数人的想法就下结论。另一半，也就是更年轻的那一半，会抱怨说这已经是旧闻了。
