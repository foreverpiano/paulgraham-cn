关于 Web 2.0 的访谈

July 2006 · 阅读原文

2006 年 7 月

（Ian Delaney 想为他的新书采访我。我同意了，条件是我可以在网上发布这些回答。）1. 我们从 dotcom 泡沫中学到了什么？（或者说，我们本应该学到什么？）

人们从那次泡沫中学到的一点是，要更批判性地判断初创公司。当时一些非常糟糕的想法也拿到了投资。而且被烧到的不仅仅是投资人，那些去这些公司工作的人也深受其害。现在大家都更加谨慎了。

这并不是说我们只应该考虑那些有详细赚钱计划的公司。那就太过谨慎了。如果你创建一个基于网页的初创公司，并且它变得非常受欢迎，你大概率能想出办法从中赚钱。几乎每一个 massively popular 的网站都做到了这一点。

“先做出受欢迎的东西，再去想怎么赚钱”的想法诞生于那次泡沫。它听起来不负责任，但确实有效。要求创始人必须有一个精心制定的赚钱计划，并非精明的商业思维。这是黑客们所说的“过早优化”。真正重要的是做出人们想要的东西。

大多数在泡沫时期失败的初创公司失败的原因是没人想要他们做的东西。也可能有一些公司在打造好东西，但因为烧钱太快而失败了。所以第二条规则是：不要乱花钱。

如果你能做出人们想要的东西，而且不乱花钱，那你就成功了 90%。

我并不是说商业部分无关紧要。只是和“做出人们想要的东西”相比，它要容易得多，因为后者实在太难了。

2. 是什么导致了当前这一波互联网初创公司的热潮？

我认为当前的初创公司热潮有几个原因。其中之一只是泡沫的阴霾已经散去。互联网代表了一次巨大的变革，其意义堪比蒸汽机的出现。既然后泡沫时代的低迷已经过去，人们对它感到兴奋是很自然的。

另一个因素是 Google。Google 让人们重新对互联网产生了兴趣，这不只是因为他们的 IPO。他们在那里聚集了一大批聪明人，并把这种兴奋感传递给了其他人。

初创公司将会变得越来越普遍，因为现在创办公司的成本已经非常低了。在我们投资的大多数初创公司里，第一年最大的开支仅仅是食物和房租。创办一家初创公司的成本只比什么都不做闲着贵一点点。而且你不用去某个写字楼的隔间里上班，而是可以和朋友一起在自己的项目上工作。如果你成功了，还能致富。总的来说，这对二十多岁的年轻人来说是非常有吸引力的前景。

3. 你认为“Web 2.0”是一个真实存在的东西，还是一个用来概括某些异质现象的方便标签？

“Web 2.0”是一个奇怪的词。它最初是一个会议的名称，但组织会议的人其实并不清楚它具体指什么。他们主要觉得这个名字听起来很吸引人。然而，“Web 2.0”后来逐渐有了自己的含义。互联网上出现了一些有趣的新趋势，而这类短语的特性就是会粘附在这些趋势上。

这有点像他们把名字印在一个粘性标签上，扔到地板上，结果它粘在了一个路过的人的鞋跟上。这个名字有点假，但这个人却是真实的。

4. 我收集到的意见从“互联网已经还给了人民”到“这里面没有任何新东西”都有。你怎么看？

我认为互联网上确实正在发生一些真正新的事物。如果没有才奇怪。每个技术领域都在不断出现新事物。现在有很多聪明人在研究互联网相关的东西，他们自然会提出新的想法。

和 1998 年相比，现在很多地方都不同了。网站的外观不同了，初创公司的运作方式不同了，人们使用网络的方式也不同了。这些变化是渐进的，但如果渐进的变化达到足够大的程度，它就开始变成一个不同的世界。

5. 在一个网络初创公司中，你会寻找哪些能说服你投资的因素？

我们会在初创公司创始人身上寻找两样东西：头脑和决心。在过去一年里我们学到的一点是，决心比我们之前认为的更重要，而头脑的重要性则相对较低。创始人不能太笨，但只要超过某个门槛，最重要的就是决心。

设计感也是一个很大的优势。大公司对待设计的态度几乎像是事后可以随便喷上去一样。即使是 Google 也有这个问题；他们最大的弱点是，对黑客的定义里没有包含足够大的设计成分。一个有设计感的黑客是非常危险的，尤其是作为初创公司创始人。

我们不太在意最初的想法，除非它能证明创始人的头脑和决心。想法是会变的。最重要的是创始人真的很想做一家初创公司。

好消息是，一群相当聪明且超级有决心的创始人有很大的成功机会。现在创办初创公司的成本如此之低，你有时间可以单纯靠意志力取得成功。你不需要第一次就做对。你可以经历好几次迭代。

6. 预计互联网广告支出将大幅增加。“长尾”能否从中获利？

要通过广告赚钱，你要么自己有一套卖广告的方案，要么使用类似 AdSense 的东西。不幸的是，AdSense 当前基于关键词的模式并不适合所有初创公司。但这类收入在最早的阶段可能会有点帮助。它可能不足以让一家初创公司盈利，但足以延长他们最初的资金使用时间，从而争取到更多时间。

7. 当前的趋势被大量炒作。这些是泡沫的征兆吗？

不，我不认为这是泡沫。风险投资公司现在投资的公司远没有 1999 年他们资助的那些那么可笑。那时候很多公司看起来像是故意在搞 parody。

当然存在大量炒作。比如，有很多网站使用廉价的“Web 2.0”设计元素来显得很酷。所有那些渐变效果、“Beta”标签和大号字体在几年后都会显得非常过时。但糟糕的设计并不会制造泡沫。衡量泡沫的标准是投资，而这方面目前仍处于可控范围。

8. 你认为哪些想法、价值观或方法会从 Web 2.0 永久地保留下来，并延续到 Web 3.0 带来的未来？

我怀疑会不会有 Web 3.0 这种东西。我认为会有太多人用“Web 3.0”这个词来描述他们关于网络未来的个人理论，以至于它会失去所有可信度。等到真正出现大到值得用这个名字的变化时，已经没人愿意用它了。“Web 2.0”已经快接近可信度的边缘了。我认识的人里很少有人能严肃地使用它。“Web 3.0”可能已经死了。

但回到根本问题，是的，确实有一些我认为会永久存在的趋势。其中之一是对用户的日益重视。这里有几种有前景的变体。最明显的是社交网络网站，它们完全围绕用户展开。但也有更微妙的变体——比如通过投票决定头条新闻的新闻网站，像 Digg 和 Reddit；还有用户发布自己内容的网站，像 Blogger 和现在的 YouTube。这种“内容”目前被称为“user-generated content”，但如果它成为默认模式，可能会有一个更简短的名称。

另一个会持续下去的趋势是基于网页的软件。这在九十年代就开始了，但现在你可以做到的事情多得多，以至于每个人都能看清这就是未来——甚至包括 Microsoft。我认为二十年后人们使用的大部分软件都将在服务器上运行。

还有一个会持久的社会趋势：初创公司世界将越来越多地由技术人员而不是商业人士主导。正如在许多其他领域一样，Google 是未来的模板。黑客主导着 Google，这就是 Google 能胜出的原因。

泡沫时期很多最具代表性的蹩脚初创公司之所以那样，正是因为它们是由商业人士创办的，然后这些人再去找黑客来实现他们的想法。这个模式在 1960 年可能行得通，但在 1998 年已经不太有效了，而且每一年都在变得更加过时。我认为未来属于黑客。技术在商业中的占比越来越大，所以权力自然会转移到这方面的专家手中，而不是管理或金融人员。
